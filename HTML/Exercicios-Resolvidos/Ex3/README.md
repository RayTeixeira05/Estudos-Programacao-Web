# Exercício 3 – Tabela

## Objetivo
Criar uma tabela exibindo informações de alunos, incluindo:

Nome
Nota 1
Nota 2
Média

## Tecnologias
- HTML5

## Descrição
A tabela deve ser construída utilizando as seguintes tags:

<table> → cria a tabela
<thead> → representa o cabeçalho da tabela (títulos das colunas)
<tbody> → representa o corpo da tabela (dados)
<tr> → define uma linha
<th> → define cabeçalho
<td> → define dados
